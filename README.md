# Chemical Space Analysis of RNA-Small Molecule Interactions

This repository contains the analysis code and processed data for the manuscript:
**"[Physicochemical and RNA Structural Determinants of Selective RNA Binding by Drug-Like Small Molecules]"**

## Repository Structure

```
.
├── code/                          # Analysis scripts
│   ├── UMAP-plot.ipynb           # Chemical space UMAP visualization
│   ├── Chemical_descriptors.ipynb # Physicochemical property analysis
│   ├── Chemical_descriptors_Selectivity-based.ipynb  # Selectivity comparison
│   ├── Rader_chart_Selectivity-based.ipynb  # Radar chart visualization
│   ├── Hit_rate.ipynb            # Hit rate analysis
│   ├── RNA_property.ipynb        # RNA structural property analysis
│   ├── Length_RNA-library.ipynb  # RNA library length distribution
│   ├── Substructure_list.ipynb   # Chemical substructure analysis
│   ├── Swarm_Plot_binding_site.ipynb  # Binding site visualization
│   └── binding_site_statistical_analysis.py  # Fisher's exact test for binding sites
│
├── data/                          # Data files
│   ├── 0_input/                  # Input data
│   ├── input_for_KNIME_cpd/      # Compound data for KNIME workflow
│   ├── input_for_KNIME_rna/      # RNA data for KNIME workflow
│   ├── output_from_KNIME/        # Processed data from KNIME
│   └── output_from_code/         # Statistical analysis results
│
└── mol_images/                    # Molecular structure images
```

## Analysis Pipeline

### 1. Chemical Space Visualization
```bash
jupyter notebook code/UMAP-plot.ipynb
```
- UMAP dimensionality reduction using 12 molecular descriptors
- Parameters: n_neighbors=5, min_dist=0.8, spread=3.0, random_state=123
- Descriptors: MW, LogP, TPSA, BertzCT, Fsp3, NumAromaticRings, HBD, HBA, RotBonds, NumHeteroatoms, RingCount, MolMR

### 2. Physicochemical Property Analysis
```bash
jupyter notebook code/Chemical_descriptors.ipynb
jupyter notebook code/Chemical_descriptors_Selectivity-based.ipynb
```
- Violin plots and bar plots for molecular descriptors
- Mann-Whitney U test with Cliff's delta effect size

### 3. Binding Site Analysis 
```bash
python code/binding_site_statistical_analysis.py
jupyter notebook code/Swarm_Plot_binding_site.ipynb
```
- Fisher's exact test for RNA structural motif enrichment
- Chi-square test for overall distribution comparison

### 4. RNA Library Characterization
```bash
jupyter notebook code/RNA_property.ipynb
jupyter notebook code/Length_RNA-library.ipynb
```

## Data Description

### Input Data
| File | Description |
|------|-------------|
| `chem-lib_FDAandInternal.csv` | Compound library with SMILES (FDA-approved, R-BIND2.0 and internal libaray) |
| `input_for_KNIME_rna.csv` | Target RNA information containing RNA structure generated by RNA_property.ipynb (input for KNIME workflow) |
| `rna_library.csv` | target RNA library as input for RNA_property.ipynb (mock data) |

### Input for KNIME
| File | Description |
|------|-------------|
| `Internal_cpd_library.table` | Internal compound library with SMILES and hit information |
| `R-BIND2_cb2c00224_si_001.xlsx` | R-BIND 2.0 reference compounds |
| `FDA_approved_Drug_drugatfda_BindingDB_2D.sdf` | FDA-approved drugs from BindingDB |


### Output Data (KNIME)
| File | Description |
|------|-------------|
| `RNA_length.csv` | Length of RNA in RNA library library  |
| `RNA_motif.xlsx` | Distribution of the number of RNA structure in RNA library  |
| `BindingScore_vs_selectivity.png` | Scatter plot for binding score and selectivity generated by KNIME workflow  |
| `chemical-space.csv` | Compound info for UMAP used as input of Chemical_descriptors.ipynb which was made by KNIME workflow|
| `hit_selective_comparison.csv` | Selective vs non-selective comparison with binding score (used for Chemical_descriptors_Selectivity-based.ipynb) |
| `binding-site.csv` | Binding site classification data for hit compound which was used for Swarm_Plot_binding_site.ipynb as a input file|
| `Img-substructure_hit_rate.xlsx` | Substructure image and hit rate with 95% CI made by KNIME workflow |
| `hit_substructure.csv` | Substructure SMILES and hit rate with 95% CI made by KNIME workflow |

### Output Data (python)
| File | Description |
|------|-------------|
| `umap_results.csv` | Compounds for UMAP analysis |
| `S1_binding_site_test_results.csv` | Fisher's exact test results made by binding_site_statistical_analysis.py |
| `S2_Statistical_Analysis_Complete.xlsx` | Complete statistical analysis generated by Chemical_descriptors.ipynb |
| `S3_Selectivity_Statistical_Analysis.xlsx` | Selectivity statistical analysis generated by Chemical_descriptors_Selectivity-based.ipynb |
